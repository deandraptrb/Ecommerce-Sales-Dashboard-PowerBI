## Overview

This is an interactive **E-Commerce Sales Dashboard** built in Power BI. It provides clear insights into revenue, quantity sold, customer spending, product performance, and geographical distribution.

This project was created while following the tutorial by **Freedom Oboh**.

---

## Key Features

- Revenue, Quantity & Avg Unit Price KPIs with YoY comparison
- Monthly Sales Trend
- Top 5 Customers by Spending
- Top 5 Products by Quantity Sold
- Geographical Sales Distribution

## How to Use

1. Download the `Ecommerce Dashboard.pbix` file
2. Open it with **Power BI Desktop**
3. Click **Refresh** to load the data (if needed)
